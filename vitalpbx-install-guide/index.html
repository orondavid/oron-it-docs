    מדריך התקנת VitalPBX על שרת לינוקס

    דרישות מוקדמות

    • מחשב/שרת עם גישה לאינטרנט
    • כתובת IP קבועה
    • שם מתחם (לא חובה, אך מומלץ)
    • הפצת לינוקס נתמכת: VitalPBX מבוססת על CentOS 7 – לכן נשתמש בגרסה זו

    חשוב: אין להתקין VitalPBX על מערכת עם שירותים קיימים או שימושים נוספים.

    שלב 1: התקנת CentOS 7

    1. הורד את ISO של CentOS 7:

    2. https://www.centos.org/download/

    3. התקן את CentOS 7 על השרת (מומלץ מצב Minimal Install)

    4. הגדר כתובת IP סטטית ב־/etc/sysconfig/network-scripts/ifcfg-eth0 לדוגמה:

    BOOTPROTO=static
IPADDR=192.168.1.100
NETMASK=255.255.255.0
GATEWAY=192.168.1.1
DNS1=8.8.8.8
ONBOOT=yes

    הפעל מחדש את השירות:

    systemctl restart network

    שלב 2: עדכון מערכת והתקנת תלות בסיסית

    yum update -y
yum install -y epel-release nano wget curl net-tools

    שלב 3: התקנת VitalPBX

    wget https://repo.vitalpbx.org/vitalpbx/v4/installer/vitalpbx-install
chmod +x vitalpbx-install
./vitalpbx-install

    ההתקנה תמשך מספר דקות ותתקין את כל רכיבי המערכת הדרושים (Asterisk, MariaDB, Apache ועוד).

    שלב 4: גישה לממשק הניהול

    לאחר סיום ההתקנה, פתח דפדפן וגש לכתובת ה-IP של השרת:

    http://כתובת-שרת

    • לדוגמה: http://192.168.1.100

    • עקוב אחר אשף ההגדרות הראשוני לקביעת:

    • סיסמת אדמין
    • אזור זמן
    • פרטי רשת
    • תצורה בסיסית למערכת

    שלב 5: יצירת שלוחה לבדיקה

    1. התחבר לממשק הניהול של VitalPBX.
    2. בתפריט הצד, עבור אל PBX > Extensions
    3. לחץ על Add Extension
    4. בחר PJSIP כטכנולוגיה
    5. מלא את השדות הבאים:
    6. Extension: 1000
    7. Name: Test User
    8. Secret: סיסמה חזקה (לדוגמה: Test@1234)
    9. יתר ההגדרות ניתן להשאיר כברירת מחדל
    10. לחץ על Save ולאחר מכן על Apply Changes בצד ימין למעלה

    שלב 6: חיבור תוכנת טלפון (Softphone)

    השתמש באפליקציה כמו Zoiper או MicroSIP:

    • Username: 1000
    • Password: הסיסמה שהגדרת
    • Domain / Host: כתובת ה-IP של השרת (לדוגמה 192.168.1.100)
    • Transport: UDP

    לאחר החיבור, אם המערכת פעילה והשלוחה תקינה – תופיע כ־"Registered" באפליקציה.

    שלב 7: חיבור קו SIP Trunk מול ספק 019

    1. הכן את הפרטים שקיבלת מהספק 019:
    2. שם משתמש (Username)
    3. סיסמה (Password)
    4. כתובת שרת SIP (לרוב sip.019tel.co.il)

    5. פורט (לרוב 5060)

    6. גש ל־PBX > External > Trunks

    7. לחץ על Add Trunk
    8. בחר PJSIP
    9. מלא את השדות הבאים:
    10. Description: 019 Trunk
    11. Username: כפי שסופק
    12. Password: כפי שסופק
    13. SIP Server: sip.019tel.co.il
    14. SIP Port: 5060
    15. Authentication: כן (Username + Password)
    16. Transport: UDP
    17. לחץ על Save ולאחר מכן על Apply Changes

    חשוב: ודא שפורט 5060 פתוח ב־firewall ושכתובת ה־IP הציבורית שלך רשומה אצל 019.

    שלב 8: הגדרת שיחות נכנסות (Inbound Route) לדוגמה

    1. עבור אל PBX > Call Routing > Inbound Routes
    2. לחץ על Add Inbound Route
    3. מלא את השדות:
    4. Description: 019 Incoming
    5. DID Pattern: השאר ריק או הכנס את המספר שסופק ע"י 019
    6. CID Pattern: ריק
    7. תחת Destination:
    8. בחר Extensions
    9. ואז בחר את השלוחה 1000
    10. לחץ Save ואחר כך Apply Changes

    כעת כל שיחה נכנסת מהקו של 019 תנותב ישירות לשלוחה 1000.

    שלב 9: הגדרת שיחות יוצאות (Outbound Route) לדוגמה

    1. עבור אל PBX > Call Routing > Outbound Routes
    2. לחץ על Add Outbound Route
    3. מלא את השדות:
    4. Description: 019 Outgoing
    5. Dial Pattern: 0. (כל מספר שמתחיל ב־0)
    6. תחת Trunk Sequence:
    7. בחר את ה־Trunk של 019 שיצרת קודם
    8. לחץ Save ואחר כך Apply Changes

    כעת כל שלוחה שתחייג מספר שמתחיל ב־0, תנותב החוצה דרך ספק 019.

    שלב 10: ביצוע גיבוי והקשחת אבטחה

    1. צור snapshot אם מותקן ב־VM
    2. הגדר סיסמאות חזקות למשתמשים
    3. שנה את פורט הגישה לממשק הניהול אם נדרש (כדי להימנע מסריקות אוטומטיות)
    4. ודא שמוגדר חומת אש (firewalld או iptables) המגבילה גישה לפי IP

    שלב 11: הגדרות נוספות (טלפוניה)

    • יצירת SIP Trunks נוספים
    • יצירת תורים (Queues)
    • הגדרת הקלטות, IVR ועוד
