    📘 מדריך: איך להפעיל קוד VBA באקסל (ולשלב כפתור בגיליון)

    מדריך זה מסביר כיצד להוסיף ולהריץ קוד VBA באקסל, כולל יצירת כפתור להפעלה ישירה של המאקרו, טיפים חשובים והערות אבטחה. מתאים לַּגרסאות Excel 2016 ומעלה (במערכות Windows 10/11).

    📀 דרישות מוקדמות

    • Microsoft Excel מותקן ( לא גרסת Web )
    • קובץ בפורמט *.xlsm (מאפשר מאקרו)
    • הרשאות הפעלת מאקרו ו-VBA

    🧱 שלבים להפעלת קוד VBA

    1. פתיחת עורך הקוד (VBA Editor)

    • פתח את הקובץ הרצוי
    • לחץ Alt + F11 כדי להכנס לעורך VBA

    2. הוספת מודול חדש

    • בחלון השמאלי (Project Explorer):

    • לחץ ימין על שם הקובץ

    • בחר: Insert > Module

    3. הדבקת קוד לדוגמא

    Sub ShowHello()
    MsgBox "Hello from your macro!"
End Sub

    4. שמירת הקובץ עם תמיכה במאקרו

    • עבור לּ File > Save As
    • בחר סוג קובץ: Excel Macro-Enabled Workbook (*.xlsm)

    🔘 הוספת כפתור להפעלת המאקרו מתוך הגיליון

    1. עבור לגיליון בוו אתה רוצה לשום בו את הכפתור
    2. בתפריט העליון עבור לּ Developer > Insert > Button (Form Control)
    3. לחץ וגרור על הגיליון ליצירת כפתור
    4. בחר את שם המאקרו (למשל: ShowHello) ולחץ OK
    5. לחץ ימין על הכפתור → Edit Text → שנה לטקסט כגון: ▶ הפעל מאקרו
    6. יש אפשרות להפעיל את המקרו דרך - Alt + F8

    🔐 אם המאקרו חסום (Security Warning)

    1. סגר את הקובץ
    2. לחץ עליו ימין → Properties
    3. סמן V ליד Unblock → Apply → OK
    4. פתח שוב ולחץ Enable Content בראש הגיליון

    🛠️ איך להוסיף את לשונית Developer אם היא נראה

    1. File > Options > Customize Ribbon
    2. סמן ✅ ליד Developer
    3. לחץ OK

    💡 טיפים מתקדמים

    • להפעלת קוד אוטומטית בפתיחת הקובץ:
    Private Sub Workbook_Open()
    MsgBox "ברוך הבא לקובץ!"
End Sub

    • (קוד זה נכנס בתחת ThisWorkbook, לא במודול רגיל)

    • אפשר להוסיף גיליונות עם קוד שנגשר משתמשים

    • הוסף של UserForm לממשק גרפי

    🧪 דוגמה נוספת – קיבוץ שורות לפי ערך בעמודה

    Sub GroupRowsByColumn()
    Dim ws As Worksheet
    Dim lastRow As Long
    Dim colIndex As Integer
    Dim colLetter As String
    Dim currentVal As Variant
    Dim startRow As Long
    Dim i As Long

    Set ws = ActiveSheet

    colLetter = InputBox("Enter column letter to group by (e.g. A):")
    If colLetter = "" Then Exit Sub

    colIndex = Range(colLetter & "1").Column
    lastRow = ws.Cells(ws.Rows.Count, colIndex).End(xlUp).Row

    ' מיון לפי העמודה שנבחרה
    ws.Sort.SortFields.Clear
    ws.Sort.SortFields.Add Key:=Range(colLetter & "2:" & colLetter & lastRow), _
        SortOn:=xlSortOnValues, Order:=xlAscending
    With ws.Sort
        .SetRange ws.Range("A1").CurrentRegion
        .Header = xlYes
        .Apply
    End With

    Application.ScreenUpdating = False
    On Error Resume Next

    startRow = 2
    currentVal = ws.Cells(startRow, colIndex).Value

    For i = startRow + 1 To lastRow + 1
        If ws.Cells(i, colIndex).Value <> currentVal Or i = lastRow + 1 Then
            If i - startRow > 1 Then
                ws.Rows(startRow + 1 & ":" & i - 1).Group
            End If
            startRow = i
            currentVal = ws.Cells(i, colIndex).Value
        End If
    Next i

    ws.Outline.ShowLevels RowLevels:=1
    Application.ScreenUpdating = True

    MsgBox "Grouping complete."
End Sub
