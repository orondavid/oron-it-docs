# הסבר לפקודות התקנת Ubuntu Server

להלן הסברים לפקודות שבהן נעשה שימוש במדריך:

---

## 🔹 `ip a`

- מציג את כתובות ה-IP של המחשב
- חשוב כדי לדעת מה הכתובת להתחברות ב־SSH

---

## 🔹 `sudo apt update && sudo apt upgrade -y`

- `sudo` – ביצוע כ־root
- `apt update` – רענון רשימת החבילות
- `apt upgrade -y` – שדרוג אוטומטי של כל החבילות

---

## 🔹 `sudo apt install openssh-server -y`

- מתקין את שרת SSH
- מאפשר התחברות מרחוק דרך SSH

---

## 🔹 `sudo systemctl status ssh`

- מציג את מצב שירות SSH
- האם הוא רץ? מופעל באתחול? שגיאות?

---

## הערות

- כל הפקודות נכתבות בטרמינל לאחר התחברות
- כל פקודה שמתחילה ב־`sudo` דורשת סיסמת משתמש
